# 数据库迁移工具

一个基于Vue 3 + Vite + Element Plus的数据库迁移工具，支持MySQL和SQL Server数据库之间的结构和数据迁移。

## 功能特性

### ✅ 已完成功能
- **数据源管理**: 支持MySQL和SQL Server数据库连接配置
- **迁移任务**: 向导式迁移流程，包含以下步骤：
  - 选择来源和目标数据库
  - 选择需要迁移的表
  - 选择迁移类型（结构迁移/数据迁移/结构和数据）
  - 实时显示迁移进度和日志
- **进度监控**: 可视化进度条和详细的迁移日志
- **用户界面**: 现代化的UI设计，支持响应式布局

### 🚧 开发中功能
- 数据源管理页面
- 系统设置页面
- 迁移历史记录
- 批量迁移任务管理

## 技术栈

- **前端框架**: Vue 3
- **构建工具**: Vite

## 使用说明

### 1. 配置数据源
- 点击"添加数据源"按钮
- 选择数据库类型（MySQL或SQL Server）
- 填写连接信息（主机、端口、数据库名、用户名、密码）
- 点击"测试连接"验证配置
- 点击"保存配置"完成数据源设置

### 2. 创建迁移任务
- 在"迁移任务"标签页中，按照向导步骤操作：
  - **步骤1**: 选择来源数据库和目标数据库
  - **步骤2**: 选择需要迁移的表
  - **步骤3**: 选择迁移类型
  - **步骤4**: 执行迁移任务

### 3. 监控迁移进度
- 实时查看迁移进度条
- 查看详细的迁移日志
- 支持导出迁移日志

## 支持的迁移类型

- **结构迁移**: 仅迁移表结构，不包含数据
- **数据迁移**: 迁移表结构和所有数据
- **结构和数据**: 同时迁移表结构和数据

## 数据库支持

### MySQL
- 版本: 5.7+
- 默认端口: 3306
- 支持所有MySQL数据类型

### SQL Server
- 版本: 2016+
- 默认端口: 1433
- 支持所有SQL Server数据类型
